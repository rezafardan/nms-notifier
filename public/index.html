Pastikan konfigurasi berikut sudah dilakukan:

1. Edit grafana.ini di server Grafana:
   [security]
   allow_embedding = true
   cookie_samesite = disabled
2. Restart service Grafana
3. Login ke Grafana di browser ini terlebih dahulu
4. Isi GRAFANA_API_KEY di file .env
5. Restart npm start
